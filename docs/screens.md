# 画面設計

## 画面一覧

```mermaid
flowchart TD
    A[ログイン] --> B[ダッシュボード\nカンバンボード]
    B --> C[エピソード詳細]
    B --> D[ゲスト管理]
    B --> E[新規エピソード作成]
    C --> F[ショーノートエディタ]
    C --> G[音声アップロード]
    D --> H[ゲスト詳細]
```

## S-1: カンバンボード（メイン画面）

```mermaid
graph LR
    A[カンバンボード] -->|カード\nクリック| B[エピソード詳細]
    A -->|＋ボタン| C[作成モーダル]
    C -->|作成| A
    B -->|自動生成| D[ショーノート\nエディタ]
    B -->|アップロード| E[音声ファイル\n管理]
    A -->|ナビ| F[ゲスト管理]
    F -->|選択| G[ゲスト詳細]
    B -->|一括配信| H[配信確認]
```
